## About This Repository

This repository contains the weekly lab reports produced during the Red Team Training program at Cyberster.
Each report documents the tasks performed, tools used, commands executed, screenshots as proof of work, and findings mapped to the MITRE ATT&CK framework.

All activities were conducted in an isolated local lab environment using VirtualBox.
No real-world systems were targeted at any point.
